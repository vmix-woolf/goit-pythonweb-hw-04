## Домашня робота №4  
**Тема:** Асинхронне сортування файлів за розширенням

### Опис
Скрипт читає всі файли у вказаній користувачем вихідній папці та асинхронно копіює їх у цільову директорію, розподіляючи по підпапках відповідно до розширення файлів.  
Для ефективності використовується `asyncio` та `aiofiles`.

### Використання
Запуск із термінала:
```bash
python async_sorter.py --source <шлях_до_вихідної_папки> --output <шлях_до_папки_призначення>
```
#### Приклад:
```python async_sorter.py --source ./test_data --output ./sorted_data```

### Логування
Усі події записуються у файл sorter.log та дублюються в консоль.
